## What's inside

| Mechanism | Description |
|---|---|
| **Chunk Carousel** | Ring buffer 50 KB, division on overflow |
| **Markers `/`** | Hierarchical markup (word/phrase/sentence/paragraph) |
| **Coherence θ=1.987** | Complex coherence with phase |
| **Mitosis / Apoptosis** | Division and cleanup of chunks |
| **TAT-DIFF** | State delta (99.99% token savings) |
| **Holographic memory** | Recovery from 42% of data (principle) |
