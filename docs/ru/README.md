# 🧪 TAT-DEMO

## Что внутри

| Механизм | Описание |
|---|---|
| **Карусель чанков** | Кольцевой буфер 50 КБ, деление при переполнении |
| **Маркеры `/`** | Иерархическая разметка (слово/фраза/предложение/абзац) |
| **Когеренция θ=1.987** | Комплексная когеренция с фазой |
| **Митоз / Апоптоз** | Деление и очистка чанков |
| **TAT-DIFF** | Дельта состояний (экономия 99.99%) |
| **Голографическая память** | Восстановление из 42% данных (принцип) |
